### Index

* 图
    * 简单图：两个定点只有一条边相连，一条边只连接两个不同的顶点。
    * 正则图：所有顶点的度都相等的图。
    * 完全图：每一对不同顶点的都有边相连的图。
    * 路径：一个顶点到另一个顶点的邻接顶点序列。
    * 环：一条路径上所有的边都互不相同为简单路径。闭合的简单路径为环。
    * 森林：无环但不连通的图为森林。
    * 树：连通无环的图为树。
